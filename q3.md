# Problem: Fibonacci Series Generation

## Description:

Given a number `n`, your task is to generate the first `n` terms of the Fibonacci series.


## Example:
**Input:**
```
n = 6
```
**Output:**
```
0, 1, 1, 2, 3, 5
```

## Test Cases:
| Input | Output         |
|-------|----------------|
| 1     | 0              |
| 2     | 0, 1           |
| 5     | 0, 1, 1, 2, 3  |
| 7     | 0, 1, 1, 2, 3, 5, 8 |



**Final Output:**
```
0, 1, 1, 2, 3, 5
```
