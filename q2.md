# Problem: Factorial Calculation

## Description:

Given a non-negative integer `n`, your task is to write an algorithm to compute its factorial.


## Example:
**Input:**
```
n = 5
```
**Output:**
```
5! = 120
```

## Test Cases:
| Input | Output |
|-------|--------|
| 0     | 1      |
| 1     | 1      |
| 3     | 6      |
| 5     | 120    |
| 7     | 5040   |

**Final Output:**
```
120
```
